## 描述

[Nest](https://github.com/nestjs/nest) 框架 TypeScript 初始仓库。

## 安装

```bash
$ pnpm install
```

## 运行应用

```bash
# 开发模式
$ pnpm run start

# 监视模式
$ pnpm run start:dev

# 生产模式
$ pnpm run start:prod
```

## 测试

```bash
# 单元测试
$ pnpm run test

# e2e 端到端测试
$ pnpm run test:e2e

# 测试覆盖率
$ pnpm run test:cov
```
